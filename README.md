# Organiser

Dynamically build packing checklists and to-do lists

## Requirements

- Python 3.9
- Redis
- pip

## Installation

`pip install -r requirements.txt`

## Usage

### To start

This guide assumes Redis is already running

`python app.py`
Then navigate to http://0.0.0.0:8080/
